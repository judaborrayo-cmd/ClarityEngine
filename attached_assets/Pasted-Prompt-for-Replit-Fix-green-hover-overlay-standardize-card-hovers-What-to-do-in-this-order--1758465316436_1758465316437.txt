Prompt for Replit — Fix green hover overlay + standardize card hovers

What to do (in this order):

Remove blend/backdrop side effects

Search the codebase for: mix-blend-, backdrop-, bg-emerald-*/[0., ::before, ::after on hover.

Delete any mix-blend-multiply, mix-blend-color, backdrop-blur, backdrop-brightness from hover overlays. We will use plain opacity instead.

Clip & isolate each card

On every card that has a gradient hover (Services, Case Studies, Featured On, Testimonials), ensure the root has:

relative isolate overflow-hidden rounded-2xl border bg-white shadow-sm

transition + subtle hover:shadow-md

isolate creates a new stacking context so overlays don’t tint siblings.

overflow-hidden and border-radius: inherit (via rounded-[inherit]) keep the overlay inside the card.

Use a safe overlay recipe

Add a non-interactive absolute layer that fades in on hover, with no blend modes:

pointer-events-none absolute inset-0 rounded-[inherit] opacity-0 group-hover:opacity-100

A light gradient or single-color tint with low opacity (e.g. opacity-[.08] to .12 max).

Layering

Keep overlay at z-0.

Put content wrapper at relative z-10.

If any link/button becomes hard to read on hover, bump its color explicitly on group-hover:.

Kill any global hover styles

Ensure there is no global rule like *:hover { background: ... } or .group:hover * { ... } that could be painting large areas.

Consistency

Apply the same “card” recipe to: Service tiles, Case Study cards, Featured On rows, and Testimonial cards.

Drop-in code (Tailwind + React pattern)

A. Reusable Card wrapper (safe hover gradient)

// Card.jsx
export default function Card({ className = "", children }) {
  return (
    <div className={`group relative isolate overflow-hidden rounded-2xl border border-neutral-200 bg-white shadow-sm transition hover:shadow-md ${className}`}>
      {/* Hover overlay (no blends) */}
      <div className="pointer-events-none absolute inset-0 z-0 rounded-[inherit] opacity-0 group-hover:opacity-100 transition duration-300 bg-gradient-to-br from-violet-500/10 via-indigo-500/10 to-emerald-500/10" />
      {/* Content */}
      <div className="relative z-10">
        {children}
      </div>
    </div>
  );
}

Why this fixes the “green wash”

The unwanted tint happens when a blending/overlapping layer isn’t clipped or inherits the page backdrop. isolate + overflow-hidden + no blend modes confines color to the card only. Using plain opacity on a subtle gradient keeps your brand look without bleeding into neighbors.
