Prompt for Replit – Replace the Arrow With Paired Comparison

Please replace the current Before/After block (with the purple arrow) with a paired comparison layout:

Goals

Emphasize contrast by aligning each “Before” pain with the specific “After” fix in the same row.

No arrows. Use a small “VS” pill (desktop) or “Fix” chip (mobile) between columns.

Use soft tints: rose for Before, emerald for After.

Keep the headline/subheadline exactly as they are:
“Imagine doubling your bookings without doubling your ad spend.” / “That’s the growth engine we build for ambitious brands.”

Design

Two columns on desktop; stacked on mobile.

Left column header chip: Before (rose), right column header chip: After (emerald).

Each row is a rounded card with an icon + text. Left uses AlertTriangle, right uses CheckCircle2.

Subtle dotted vertical divider between columns on desktop with a small VS pill centered.

Smooth fade/slide on scroll; quick crossfade when resizing.

No fabricated numbers—if we ever show metrics deltas, they must be real; otherwise omit.

Content Pairs (use these)
Before ↔ After

Wasted budget on vanity metrics ↔ Campaigns tied to revenue outcomes

No clear roadmap or milestones ↔ Clear funnel roadmap with milestones

Broken tracking hides true ROI ↔ Clean tracking that connects ad spend to revenue

Cookie-cutter execution ↔ Custom strategy aligned to goals

A11y & polish

Keyboard focus rings, semantic headings.

Generous spacing (py-16, gap-6/8), rounded-2xl, soft shadows.

Drop-in React Component (Tailwind + lucide + optional Framer Motion)

Why this works

It maps pain → fix one-to-one, so the transformation is obvious without guessing what an arrow means.
