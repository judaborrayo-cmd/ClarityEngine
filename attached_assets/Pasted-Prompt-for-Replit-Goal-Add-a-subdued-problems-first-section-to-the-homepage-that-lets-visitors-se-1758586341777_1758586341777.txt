Prompt for Replit

Goal: Add a subdued, problems-first section to the homepage that lets visitors self-categorize by avatar and dynamically shows the relevant problems/solutions for that avatar.

What to build:

Create a new React component named ClarityEngineAvatarProblems using Tailwind (no external libraries).

The component must include:

A section title: “Problems We Eliminate”.

An avatar selector with three options:

Revenue Protectors — “Worried your ad dollars are being wasted.”

Growth Drivers — “Campaigns have stalled and you need scale.”

Conversion Optimizers — “Traffic isn’t turning into bookings or sales.”

When an avatar is selected, swap the three problem → solution cards shown below the selector (no page reload).

A simple CTA bar at the bottom with a link to #book that says: “Book a Complimentary Call”.

Style requirements (subdued to match current homepage):

Keep backgrounds light, use white cards with faint borders and very soft shadows.

Use neutral palette (slate/gray) and avoid bright colors.

Rounded corners (rounded-2xl) and gentle hover states.

No heavy icons—use small, subtle accents only (e.g., a faint corner blob).

Placement:

Insert this section below the hero (or in Testimonials page as a second instance, if desired). On homepage, it should sit near the top to set a problems-first frame.

Code to use:

Use the exact component I’ve provided (in the canvas) named ClarityEngineAvatarProblems.
