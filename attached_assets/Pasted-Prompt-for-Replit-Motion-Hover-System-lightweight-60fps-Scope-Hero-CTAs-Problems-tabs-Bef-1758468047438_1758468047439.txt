Prompt for Replit — Motion & Hover System (lightweight, 60fps)

Scope: Hero CTAs, Problems tabs, Before/After pairs, Services grid, Case Studies, Testimonials, Featured On.

Performance rules: only animate opacity + transform. Add transform-gpu and will-change: transform on animated elements. Respect prefers-reduced-motion.

1) Add small motion presets (Framer Motion)

Create components/motionPresets.ts:

import { Variants } from "framer-motion";

export const fadeUp: Variants = {
  hidden: { opacity: 0, y: 20 },
  show:   { opacity: 1, y: 0, transition: { duration: 0.45, ease: "easeOut" } }
};

export const stagger: Variants = {
  hidden: {},
  show:   { transition: { staggerChildren: 0.06, delayChildren: 0.05 } }
};


Create a tiny wrapper components/MotionSection.tsx:

import { motion, useReducedMotion } from "framer-motion";
import { PropsWithChildren } from "react";
import { stagger } from "./motionPresets";

export default function MotionSection({ children, className="" }: PropsWithChildren<{className?: string}>) {
  const reduce = useReducedMotion();
  if (reduce) return <section className={className}>{children}</section>;
  return (
    <motion.section
      className={className}
      variants={stagger}
      initial="hidden"
      whileInView="show"
      viewport={{ once: true, margin: "-100px" }}
    >
      {children}
    </motion.section>
  );
}


Use per-item animation where needed:

import { motion } from "framer-motion";
import { fadeUp } from "@/components/motionPresets";

// wrap a grid with <MotionSection>, then on each card:
<motion.div variants={fadeUp} className="...">...</motion.div>


Apply to: Services grid cards, Case Study cards, Testimonial cards, “Featured On” rows (the first appearance of each group).

2) Premium Card Hover (lift + sheen)

Add a reusable wrapper components/Card.tsx (uses your fixed overlay approach, no blend modes):

export default function Card({ className="", children }: {className?: string; children: React.ReactNode}) {
  return (
    <div className={`group relative isolate overflow-hidden rounded-2xl border border-neutral-200 bg-white shadow-sm transition
                     hover:shadow-md transform-gpu will-change-transform ${className}`}>
      {/* hover gradient tint (very subtle) */}
      <div className="pointer-events-none absolute inset-0 z-0 rounded-[inherit] opacity-0 group-hover:opacity-100 transition duration-300
                      bg-gradient-to-br from-violet-500/8 via-indigo-500/8 to-emerald-500/8" />
      {/* sheen */}
      <div className="pointer-events-none absolute -inset-y-8 -left-1/3 w-1/3 rotate-12 bg-white/0
                      group-hover:bg-white/20 transition duration-300" />
      <div className="relative z-10">{children}</div>
    </div>
  );
}


Use Card for Services, Case Studies, and Featured rows. Remove any old ::before/::after hovers on those tiles.

If you want a slightly more noticeable sheen, animate it:

3) Animated tab indicator for “Problems We Eliminate”

Use a sliding pill that follows the active tab (keeps your icons/text; just adds motion).

import { motion } from "framer-motion";
const tabs = ["Agencies","Marketing Teams","Executives","Growing Brands"] as const;
type Tab = typeof tabs[number];

export function AvatarTabs({ value, onChange }:{value:Tab; onChange:(t:Tab)=>void}) {
  return (
    <div className="relative inline-flex gap-2 p-2 rounded-2xl bg-neutral-50 border border-neutral-200">
      {/* moving indicator */}
      <div className="grid grid-cols-4 relative">
        {tabs.map((t) => (
          <button key={t}
                  onClick={() => onChange(t)}
                  className={`relative z-10 px-4 py-2 rounded-xl text-sm font-medium transition
                              ${value===t ? "text-emerald-800" : "text-neutral-700 hover:text-neutral-900"}`}>
            {value===t && (
              <motion.span
                layoutId="avatar-pill"
                className="absolute inset-0 -z-10 rounded-xl bg-emerald-50 border border-emerald-200"
                transition={{ type: "spring", stiffness: 500, damping: 40 }}
              />
            )}
            {t}
          </button>
        ))}
      </div>
    </div>
  );
}


(Just replace your current static selected style with this; keep your icons inside the button label if you like.)

4) Link underline slide (for “Learn more”, case study links, etc.)

Add this utility in your global CSS (or Tailwind @layer utilities):

.link-underline {

}
.link-underline::after {
  content:"";
  position:absolute;
  left:0; bottom:-2px; height:2px; width:100%;
  background: currentColor;
  transform: scaleX(0);
  transform-origin: 0 50%;
  transition: transform .25s ease-out;
}
.link-underline:hover::after { transform: scaleX(1); }

If you have any global CSS animations, wrap with:

@media (prefers-reduced-motion: reduce) {
  * { animation: none !important; transition: none !important; }
}
